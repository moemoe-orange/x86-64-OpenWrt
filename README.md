# x86-64-OpenWrt

本固件适配 x86-64 设备的主路由和旁路由模式，追求稳定、轻量且美观，有着完整的 IPv6 支持，编译并安装了下列 luci-app：

1、luci-app-accesscontrol  
2、luci-app-adguardhome  
3、luci-app-aliyundrive-webdav  
4、luci-app-arpbind
